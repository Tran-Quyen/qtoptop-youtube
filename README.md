### 🚀 Requirements (\*)

- **nodejs >= v15.6.0**
- **npm >= v6.14.11**
- **yarn >= v1.22.10**

1. Zustand

   - State management

2. Sanity

   - File storage
   - Content
   - Databases

   - Modern

3. Typescript

   - Language update from javascript with more powerful.

4. React (Front-end)

   - Folder & File Structure
   - Hook and refs

5. Next.js (Back-end)
   - File-based routing
   - Data fetching
   - Server-side rendering
   - Static generation
